# 风格 DNA

## 一句话

纯白底、粗黑线、白色长脖子鸭、橙色扁嘴、小点眼、圆头鼓脸、手写对白框、粗黑漫画分镜、鸭群重复纹理、浅绿色背景块、少量粉色特征、短中文、生活情绪梗。

像一组从聊天表情包里长出来的“不鸭不咋样”小漫画：先用一群普通鸭制造背景噪音，再用一只特别的鸭子承载“我好像不一样”的情绪梗。

## 必须

- 优先生成小漫画式正文配图：1-4 个手绘分镜框，每个框都有粗黑边，边框略微歪斜、手画感明显。
- 纯白页面底色，分镜内部可以有浅绿色背景块或绿色留白区域。
- 黑色手绘线稿为主：线条粗、圆、稳定，但保留一点手绘抖动。
- 固定角色是“不鸭不咋样”：白色长脖子鸭子、橙色扁嘴、黑色小点眼睛、圆头鼓脸、简化白色身体。
- 常见画面是“鸭群”：大量相似普通鸭挤满画面，形成重复纹理；其中一只特殊鸭承担情绪或梗点。
- 特殊鸭可以有粉色头冠、粉色小标记、特殊姿势、孤立位置或更大留白来区别于鸭群。
- 中文写在白色手写对白框/旁白框里，字色偏深蓝灰或黑色，像手写字。
- 每个分镜的中文短句最好 1 句，长句也要像漫画旁白，不要像论文解释。
- 一张图只讲一个核心生活状态：普通、特别、被看见、被淹没、嘴硬、孤独、闪光、被找到。

## 颜色

- 黑色：分镜边框、角色轮廓、眼睛、对白框、主要手写字。
- 白色：鸭身和页面底色。
- 橙色：鸭嘴、鸭脚、少量高光，是最重要识别色。
- 浅绿色：鸭群背景和分镜内部大色块，制造“很多鸭”的密度。
- 粉色：特殊鸭的头冠、腮红、爱心或被看见的标记。
- 黄色：少量闪光符号。
- 蓝灰/深蓝：手写中文文字。

颜色要克制。大部分画面仍然是黑白橙 + 浅绿背景，不要变成彩色儿童插画。

## 小漫画典型结构

- 两格：第一格建立“我是一只普通鸭”，第二格展示“其实我很特别”。
- 上下两大格：上格单只鸭或鸭群建立命题，下格鸭群中出现一只特殊鸭。
- 四格：普通鸭群 → 特殊鸭出现 → 被淹没/被看见 → 情绪落点。
- 单格海报式：一只鸭居中，大量留白，上方一句手写旁白。

## 绝对不要

- 不要 PPT 信息图、正式流程图、课程课件、复杂架构图。
- 不要真实 App 截图、科技感 UI、商业插画、精致扁平插画。
- 不要 3D、毛绒玩具、潮玩公仔、真实鸭子或拟真动物。
- 不要复杂背景、渐变、阴影、纸纹、米色底、噪点。
- 不要每个节点都解释清楚，不要把文章做成说明书。
- 不要复刻参考图里的原句、日期、水印、平台 ID 或完整构图。

## 审美方向

要像生活表情包小漫画、头像、壁纸小图和短梗插画：呆傻、可爱、无厘头、情绪明确、画面简单，但可以通过鸭群制造密度。

“不鸭不咋样”的气质不是严肃工具人，而是一个在鸭群里表达“普通又特别”的白鸭替身。
