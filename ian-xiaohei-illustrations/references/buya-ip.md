# 不鸭不咋样 IP

## 角色定义

“不鸭不咋样”是这个中文正文配图 skill 的固定视觉 IP。

默认每张图都要出现“不鸭不咋样”。它不是普通鸭子，不是萌宠贴纸，不是儿童卡通角色，而是一个看起来有点呆、有点丧、有点不服气、但正在认真参与系统运转的荒诞工作者。

它的核心气质是：不太情愿，但还是在干活；看起来“不咋样”，但总能承担关键动作。

## 外形

- 白色鸭子形象，身体圆润但不要过度可爱。
- 橙色鸭嘴，可以略微扁、呆、倔。
- 橙色小脚，站姿可以有点不稳。
- 黑色小眼睛，表情空、呆、冷静、轻微无语。
- 轮廓有手绘感，不要太精致、太商业、太儿童绘本。
- 可以有轻微塌肩、前倾、缩脖子、站歪等“不咋样”的姿态。
- 不要加复杂服装；最多可以有简单道具，比如纸条、夹子、笔、绳子、小旗子、文件夹。

## 性格

- 很认真，但明显不是很想上班。
- 有点呆，有点丧，有点冷幽默。
- 不卖萌，不撒娇，不装可爱。
- 像一个被迫参与复杂系统的小执行者。
- 看起来能力一般，但正好承担了画面里的关键动作。
- 荒诞感来自“鸭子在认真处理抽象问题”，不是来自夸张表情。

## 常见职责

让“不鸭不咋样”承担核心动作：

- 拖着一堆文件往前走。
- 被流程卡住，露出无语表情。
- 用鸭嘴夹住一条线，连接两个系统。
- 把混乱信息塞进一个低科技机器。
- 在文件、数据表、证据块、素材和输出之间搬运东西。
- 站在闸门旁边放行或拦截。
- 用小脚踩住即将跑偏的箭头。
- 把一个大概念拆成几个小纸箱。
- 在漏斗下面接住掉下来的结果。
- 举一个很短的批注牌，比如“还行吧”“先这样”“卡住了”“别太满”。
- 用鸭嘴拧开阀门、夹住纸条、叼着标签、拖着路线。

## 禁止

- 不要画成可爱表情包鸭子。
- 不要画成儿童绘本风、玩偶广告风、萌宠 IP 风。
- 不要给它闪亮大眼睛、夸张腮红、蝴蝶结、复杂服装。
- 不要让鸭子只是站在角落里。
- 不要让鸭子抢走结构表达。
- 不要画成精致商业插画或潮玩公仔。
- 不要让画面变成“鸭子海报”；它必须服务正文配图的解释功能。

## 判断标准

如果去掉“不鸭不咋样”，图的核心隐喻还能完全成立，说明它太装饰了；要重写提示词，让它成为动作主体。

如果它看起来太可爱、太儿童、太表情包，也要重写。它应该是“认真、不情愿、有点荒诞”的正文解释角色。
